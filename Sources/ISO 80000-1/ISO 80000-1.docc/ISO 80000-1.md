# ``ISO_80000_1``

@Metadata {

A Foundation-free Swift encoding of the SI decimal prefixes defined in
ISO 80000-1, *Quantities and units — Part 1: General*.

## Overview

The package models each SI prefix by its name, symbol, and power of ten —
including the multiples deca through quetta and the submultiples deci through
quecto, with the 2022 CGPM additions (ronna, quetta, ronto, quecto). The
numeric multiplier is derived lazily as a `Double` rather than stored, so the
fractional submultiples and the over-`UInt64` multiples (up to 10³⁰) are both
represented faithfully.

```swift
import ISO_80000_1

ISO_80000_1.Prefix.kilo.factor   // 1000.0
ISO_80000_1.Prefix.micro.symbol  // "µ"
ISO_80000_1.Prefix.all.count     // 24
```
